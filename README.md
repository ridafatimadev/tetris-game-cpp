# Tetris Game (C++)

## Overview
This project is a console-based implementation of the classic Tetris game developed in C++ using SFML as part of the Programming Fundamentals course at Air University.

## Features
- Classic Tetris gameplay
- Score tracking
- Collision detection
- Game over detection
- Keyboard controls
- SFML-based graphics

## Technologies Used
- C++
- SFML
- Object-Oriented Programming

## Learning Outcomes
This project helped strengthen my understanding of programming fundamentals, game logic, problem-solving, and working with graphics libraries in C++.
