# Tetris Game (C++)

## Overview
This is a console-based Tetris game developed in C++ using SFML as part of the Programming Fundamentals course at Air University.

## Features
- Classic Tetris gameplay
- Score tracking
- Collision detection
- Game over logic
- SFML graphics

## Technologies Used
- C++
- SFML
- Object-Oriented Programming
